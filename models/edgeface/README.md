# EdgeFace Model Folder

Drop these files into this folder before running the new pipeline:

- `edgeface_xs.onnx`

By default the scripts reuse the existing YuNet detector already present in:

- `models/yunet_mobilefacenet/face_detection_yunet_2023mar.onnx`

If you want a separate copy under this folder, pass `--yunet-model`.

Expected commands:

```powershell
python src/edgeface/trainer.py --include-raw
python src/edgeface/evaluate.py
python src/edgeface/face_detect.py --largest-face-only
```

Notes:
- This integration uses `YuNet + EdgeFace-XS`.
- `enrollment.json` is generated by `trainer.py`.
- If you later want `SCRFD + EdgeFace`, keep the recognizer files here and add a detector-specific runner.
